# Facebook Market Image Bot

هذا المشروع يشغل بوت Node.js لصفحة Facebook Page بهذا السلوك فقط:

- ترفع صورة واحدة من الداشبورد
- البوت ينشر نفس الصورة كل 8 ساعات
- نص المنشور يكون:
  - `السوق رقم 1`
  - ثم `السوق رقم 2`
  - ثم `السوق رقم 3`
- بعد كل نشر، يفحص آخر منشور كل دقيقة
- لكل تعليق جديد:
  - يعمل إعجابًا على التعليق
  - ثم يرد عليه بصيغة فريدة مثل:
    - `شكرا ربي يجيب السوق 1.1`
    - `شكرا ربي يجيب السوق 1.2`
    - `شكرا ربي يجيب السوق 2.1`

## المتغيرات

ضع هذه القيم في `.env` أو Railway:

- `FB_PAGE_ID`
- `FB_PAGE_ACCESS_TOKEN`
- `TIMEZONE`

اختياري:

- `BASE_URL`
- `STATE_DIR`
- `DATABASE_URL`
- `FB_APP_ID`
- `FB_APP_SECRET`

## التشغيل

```bash
npm install
npm start
```

ثم افتح:

```text
http://localhost:3000
```

## الدخول

- الصفحة الرئيسية `/` تعرض شاشة الكود مباشرة
- كود الدخول الحالي هو `5598`
- بعد الدخول ستذهب إلى الداشبورد

## ما الذي يفعله البوت

1. ارفع صورة من صفحة `إدارة المنشورات`
2. اضغط `تشغيل البوت`
3. إذا لم يكن هناك منشور سابق، ينشر فورًا أول منشور
4. بعد ذلك يعيد النشر كل 8 ساعات
5. يراقب فقط آخر منشور نشره البوت
6. كل دقيقة يفحص التعليقات الجديدة ويعطيها إعجابًا ويرد عليها بترقيم فريد

## صفحات الداشبورد

- `نظرة عامة`
- `التحكم في الوقت`
- `المنشور التالي`
- `المنشورات التي تمت نشرها`
- `الأشخاص المتفاعلون`
- `إدارة المنشورات`

## رفع الصورة

في صفحة `إدارة المنشورات`:

1. اختر صورة واحدة
2. اضغط `رفع الصورة`
3. سيعرض لك البوت الصورة الحالية
4. سيستخدم نفس الصورة في كل دورة نشر لاحقة

## النشر اليدوي للتجربة

يمكنك تجربة نشر فوري من:

```text
/run-once
```

## قاعدة البيانات

إذا أضفت `DATABASE_URL` فسيتم تفعيل Postgres تلقائيًا.

سيتم استعمال جدولين:

- `scheduled_posts`
- `published_posts`

في هذا الإصدار، البوت لا يعتمد على طابور نصوص يدوي، لكنه ما زال يسجل المنشورات المنشورة في `published_posts`، ويحتفظ بالحالة العامة في `state.json`.

## Railway + Postgres

1. أضف خدمة `Postgres` داخل Railway
2. افتح خدمة `facebook-3`
3. في `Variables` أضف متغيرًا باسم `DATABASE_URL`
4. اجعل قيمته من خدمة Postgres عبر Variable Reference أو انسخ قيمة `DATABASE_URL` الداخلية
5. أعد نشر الخدمة
